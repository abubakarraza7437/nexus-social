Planned: Post Analytics

GET /api/v1/analytics/posts/

Performance metrics for individual posts.

Query ParamTypeDescription
start_datedateStart of date range
end_datedateEnd of date range
platformstringFilter by platform
sort_bystringreach, engagement, date

Metrics Definitions

MetricDescription
ReachNumber of unique users who saw the post
ImpressionsTotal number of times the post was displayed
EngagementSum of likes, comments, shares, and clicks
Engagement Rate(Engagement / Reach) × 100
Click-through Rate(Clicks / Impressions) × 100
