Apps & Endpoints

Auth Core Live

User registration, JWT login/refresh/logout, password reset, email verification, account deletion. Custom JWT payload with org, role, and MFA claims.

v1 — 9 endpoints v2 — 8 endpoints
Organizations Live

Multi-tenant org management — create/list/detail, member management, email invitations, join requests with approval flow, org stats dashboard.

v1 — 14 endpoints v2 — 15 endpoints
Social Accounts Stub

OAuth connections for Facebook, Instagram, Twitter/X, and LinkedIn. Token encryption, auto-refresh. Planned: v1 endpoints for connect/disconnect/list.

v1 — coming soon
Content Stub

Posts, media uploads, approval workflows. Supports multi-platform content targeting, rich media attachments, and editor/approval state machine.

v1 — coming soon
Scheduler Stub

Recurring post schedules using cron-based rules. Queue: scheduler (priority 8). Publishes to socialos.posts.scheduled Kafka topic.

v1 — coming soon
Publisher Stub

Publishes posts to social platforms with retry logic. Highest priority Celery queue (publish, priority 10). Internal only — no public REST API.

Internal only
Analytics Stub

Metrics collection and aggregation via ClickHouse bridge. Consumes socialos.posts.published topic. Planned: reach, engagement, follower growth APIs.

v1 — coming soon
Inbox Stub

Unified inbox for DMs, comments, and mentions across all connected platforms. Consumes socialos.inbox.messages Kafka topic.

v1 — coming soon
AI Engine Stub

AI caption and hashtag generation via OpenAI and Anthropic. Celery queue: ai (priority 6, prefork pool). Planned: caption, hashtag, tone rewrite endpoints.

v1 — coming soon
Notifications Stub

In-app notifications with WebSocket push via Django Channels. Channel groups named org_<org_id>. Internal only — no public REST API.

Internal only
Automation Stub

Rule-based automation triggers for content workflows. Planned: trigger/action rule CRUD, execution history, enable/disable toggle.

v1 — coming soon
Audit Stub

Request audit log and compliance records. AuditMiddleware logs all requests. Consumes socialos.oauth.token_refreshed topic. Internal only.

Internal only
