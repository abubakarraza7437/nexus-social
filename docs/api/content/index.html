Planned: Posts CRUD

Endpoints

MethodEndpointDescription
GET/api/v1/posts/List posts with filtering and pagination
POST/api/v1/posts/Create a new post
GET/api/v1/posts/{id}/Get post details
PATCH/api/v1/posts/{id}/Update post
DELETE/api/v1/posts/{id}/Delete post

Post Data Model

FieldTypeDescription
iduuidUnique post identifier
contenttextPost text content
mediaarrayAttached media files
platformsarrayTarget platforms: ["facebook", "twitter", "instagram"]
statusstringPost status (see workflow below)
scheduled_atdatetimeScheduled publish time (null for immediate)
published_atdatetimeActual publish time
authorobjectUser who created the post
created_atdatetimeCreation timestamp
updated_atdatetimeLast update timestamp

Post Status Workflow

Posts follow a state machine for approval and publishing.

draft → pending_approval → approved → scheduled → publishing → published
rejected failed

Approval Endpoints

MethodEndpointDescription
POST/api/v1/posts/{id}/submit/Submit for approval
