Planned: Generate Caption

POST /api/v1/ai/caption/

Generate engaging captions for social media posts.

Request Body

FieldTypeDescription
topicstringTopic or subject of the post
platformstringTarget platform (affects length/style)
tonestringprofessional, casual, humorous, inspirational
keywordsarrayKeywords to include
max_lengthintegerMaximum character count

Response

{
  "captions": [
    {
      "text": "🚀 Exciting news! We just launched...",
      "character_count": 142,
      "tone": "professional"
    },
    {
      "text": "Big things are happening at Acme Corp...",
      "character_count": 156,
      "tone": "professional"
    }
  ],
  "credits_used": 2,
  "credits_remaining": 998
}

Planned: Generate Hashtags

POST /api/v1/ai/hashtags/

Generate relevant hashtags for content.

Request Body

FieldTypeDescription
contentstringPost content to analyze
platformstringTarget platform
countintegerNumber of hashtags (default: 10)
include_trendingbooleanInclude trending hashtags

Response

{
  "hashtags": [
    { "tag": "#SocialMedia", "relevance": 0.95, "trending": false },
    { "tag": "#Marketing", "relevance": 0.88, "trending": true },
    { "tag": "#ContentCreation", "relevance": 0.82, "trending": false }
  ],
  "credits_used": 1
}

Planned: Rewrite Content

POST /api/v1/ai/rewrite/

Rewrite content with a different tone or style.

Request Body

FieldTypeDescription
contentstringOriginal content
target_tonestringDesired tone
target_platformstringOptimize for platform
preserve_meaningbooleanKeep core message (default: true)
