Notifications Internal Only

In-app notifications with WebSocket push via Django Channels.

⚠️ Internal Service

Notifications are delivered via WebSocket. There is no public REST API for notifications — they are pushed to connected clients in real-time.

Architecture

ComponentDescription
WebSocket ServerDjango Channels with Redis backend
Channel Groupsorg_{org_id} — per-organization groups
DispatcherInternal service that routes notifications

WebSocket Connection

Connection URL

wss://api.socialos.io/ws/notifications/?token={jwt_token}

Authentication

The JWT token is validated on connection. The user is automatically subscribed to their organization's channel group.

Message Format

{
  "type": "notification",
  "payload": {
    "id": "notification-uuid",
    "notification_type": "post.published",
    "title": "Post Published",
    "message": "Your post was published to Twitter",
    "data": {
      "post_id": "post-uuid",
      "platform": "twitter"
    },
    "created_at": "2024-01-15T10:30:00Z",
    "read": false
  }
}

Notification Dispatcher

The dispatcher is an internal service that routes notifications to the appropriate channel groups. 

# apps/notifications/dispatcher.py

class NotificationDispatcher:
    """Routes notifications to WebSocket channel groups."""
    
    def send_to_org(self, org_id: str, notification: dict):
        """Send notification to all members of an organization."""
        channel_layer.group_send(
            f"org_{org_id}",
            {"type": "notification", "payload": notification}
        )
    
    def send_to_user(self, user_id: str, notification: dict):
        """Send notification to a specific user."""
        channel_layer.group_send(
            f"user_{user_id}",
            {"type": "notification", "payload": notification}
        )
    
    def send_to_admins(self, org_id: str, notification: dict):
        """Send notification to org admins and owners only."""
        channel_layer.group_send(
            f"org_{org_id}_admins",
            {"type": "notification", "payload": notification}
        )
