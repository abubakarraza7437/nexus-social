Planned: Connect Account

POST /api/v1/social-accounts/connect/

Initiate OAuth flow to connect a social media account.

FieldTypeDescription
platformstringPlatform identifier: facebook, instagram, twitter, linkedin
redirect_uristringOAuth callback URL

Planned: Disconnect Account

DELETE /api/v1/social-accounts/{account_id}/

Revoke OAuth tokens and remove the connected account.

Planned: List Connected Accounts

GET /api/v1/social-accounts/

List all social accounts connected to the organization.

FieldTypeDescription
iduuidAccount identifier
platformstringPlatform name
platform_user_idstringUser ID on the platform
usernamestringPlatform username/handle
display_namestringDisplay name on platform
avatar_urlstringProfile picture URL
is_activebooleanWhether tokens are valid
connected_atdatetimeWhen the account was connected
token_expires_atdatetimeWhen the access token expires

Planned: Refresh Token

POST /api/v1/social-accounts/{account_id}/refresh/

Manually trigger a token refresh (normally handled automatically).

Security Features

FeatureDescription
Token EncryptionAll OAuth tokens are encrypted at rest using AES-256-GCM
Auto-RefreshTokens are automatically refreshed before expiry via Celery tasks
Audit LoggingAll token operations are logged to the audit trail
Kafka EventsToken refresh events published to socialos.oauth.token_refreshed
